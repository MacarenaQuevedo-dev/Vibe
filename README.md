Vibe 💬

Vibe es una aplicación de mensajería instantánea desarrollada con React, inspirada en el diseño de WhatsApp Web. Este proyecto fue creado como parte de mi aprendizaje en desarrollo frontend.
===========================
Tecnologías utilizadas
===========================
React — biblioteca principal para la interfaz de usuario
React Router — manejo de navegación y rutas
Context API — manejo de estado global
CSS puro — estilos sin frameworks externos
React Icons — iconografía
LocalStorage — persistencia de sesión y preferencias

Funcionalidades
 

Inicio de sesión con validaciones
Rutas protegidas — solo accesibles con sesión iniciada
Chat funcional con envío de mensajes
Búsqueda de contactos en tiempo real
Búsqueda de mensajes dentro del chat
Panel de detalles del contacto
Panel de perfil con cambio de nombre y color de avatar
Dropdown con opciones por chat — limpiar y cerrar
Modo oscuro
Diseño responsive para mobile
